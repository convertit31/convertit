Frequently Asked Questions (FAQs)

❓ What is ConvertIt?

ConvertIt is a free, web-based toolkit offering tools for PDF, image, and document processing — all without requiring installation or login.

🔐 Are my files secure?

Yes. All file processing happens directly in your browser using JavaScript. Your files are never uploaded to a server, ensuring complete privacy and security.

💸 Is ConvertIt completely free to use?

Yes! All of our tools are 100% free to use. There are no hidden charges, subscriptions, or usage limits.

📱 Can I use ConvertIt on my mobile device?

Absolutely. ConvertIt is designed to be fully responsive and works seamlessly on mobile phones, tablets, and desktops.

📂 What types of files can I work with?

Currently, ConvertIt supports PDF, JPG, PNG, TXT, and other common file formats depending on the tool. More formats are coming soon!

❌ Why isn’t my file uploading?

Make sure the file format is supported by the tool you're using. Also, some browsers have limitations with large files — try using Chrome or Firefox for best results.

🧰 What tools does ConvertIt currently offer?

We offer tools like PDF compressor, unlock/lock, merge/split, image resizer/compressor, text utilities, and more. New tools are added regularly.
